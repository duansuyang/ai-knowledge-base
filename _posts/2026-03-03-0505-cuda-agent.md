# CUDA Agent

## 1. 项目概述

### 1.1 一句话概述
使用大规模Agentic强化学习技术自动生成高性能CUDA计算内核。

### 1.2 详细描述
该论文提出CUDA Agent框架，利用Agentic RL方法自动化CUDA内核的生成和优化。通过智能体交互和强化学习，系统能够自动发现高效的GPU计算实现。
