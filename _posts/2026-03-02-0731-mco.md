# MCO - Multi-CLI Orchestrator

## 项目概述

MCO 是一个中立的 AI 编码智能体编排层，支持同时调度 Claude Code、Codex CLI、 Gemini CLI、OpenCode 和 Qwen Code 并行执行任务，聚合结果并返回结构化输出。

## 核心功能

- **并行扇出** - 同时调度多个智能体，聚合结果
- **去重与追踪** - 收集各智能体发现，去重重复问题，追踪来源
- **共识合成** - 可选让一个智能体总结组合结果
- **无供应商锁定** - 支持任意智能体、任意 IDE

## 应用场景

- 代码审查 - 多个智能体从不同角度审查，发现更多问题
- 架构分析 - 不同智能体发现不同设计风险
- Bug 搜索 - 更广泛覆盖代码路径和边缘情况
- 重构评估 - 多角度评估影响和安全性

## 技术特点

支持从 Claude Code、Cursor、Trae、Copilot、Windsurf 或普通 shell 调用。
