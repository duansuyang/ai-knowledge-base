# OpenSandbox

## 1. 项目概述

### 1.1 项目名称
OpenSandbox

### 1.2 一句话概述
阿里巴巴开源的通用AI应用沙箱平台，支持Coding Agents、GUI Agents、Agent评估、AI代码执行和RL训练。

### 1.3 详细描述
OpenSandbox是阿里巴巴开源的通用沙箱平台，为AI应用提供多语言SDK、统一沙箱API和Docker/Kubernetes运行时。支持编码代理、GUI代理、代理评估、AI代码执行和强化学习训练等多种场景。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 多语言SDK支持
- **描述**: 提供Python、Java、Go等多种语言的SDK，开发者可以使用自己熟悉的语言接入沙箱服务
- **为什么重要**: 降低开发者门槛，扩大使用范围

#### 创新点 2: 统一的沙箱API
- **描述**: 提供统一的RESTful API接口，屏蔽底层实现细节
- **为什么重要**: 简化集成流程，提高开发效率

#### 创新点 3: Docker/Kubernetes双Runtime支持
- **描述**: 同时支持Docker单机部署和Kubernetes集群部署

## 4. 技术细节

### 4.1 核心方法/架构

#### 4.1.1 架构设计
OpenSandbox采用微服务架构，主要组件包括：
- Sandbox Server: 沙箱服务核心
- SDK Clients: 多语言客户端
- Runtime: Docker/Kubernetes运行时

### 4.2 使用示例

```python
import asyncio
from datetime import timedelta
from code_interpreter import CodeInterpreter, SupportedLanguage
from opensandbox import Sandbox
from opensandbox.models import WriteEntry

async def main() -> None:
    # 1. Create a sandbox
    sandbox = await Sandbox.create(
        "opensandbox/code-interpreter:v1.0.1",
        entrypoint=["/opt/opensandbox/code-interpreter.sh"],
        env={"PYTHON_VERSION": "3.11"},
        timeout=timedelta(minutes=10),
    )
    
    async with sandbox:
        # 2. Execute a shell command
        execution = await sandbox.commands.run("echo 'Hello OpenSandbox!'")
        print(execution.logs.stdout[0].text)
        
        # 3. Execute Python code
        interpreter = await CodeInterpreter.create(sandbox)
        result = await interpreter.codes.run("""
import sys
print(sys.version)
result = 2 + 2
result
""", language=SupportedLanguage.PYTHON)
        print(result.result[0].text)  # 4
        
        # 4. Cleanup
        await sandbox.kill()

asyncio.run(main())
```

### 7.2 综合评价

OpenSandbox是阿里巴巴开源的AI应用沙箱平台，为AI代理提供了一个安全、隔离的执行环境。项目支持多种编程语言的SDK，可以满足不同开发者的需求。支持Docker和Kubernetes两种部署方式，从个人开发者到企业级应用都能很好地支持。对于正在开发AI代理、RL训练系统或需要安全代码执行环境的开发者来说，这是一个非常实用的工具。
