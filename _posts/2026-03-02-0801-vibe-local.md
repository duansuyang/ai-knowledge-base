# vibe-local - Free AI Coding Environment

## 项目概述

vibe-local 是一个免费的本地 AI 编程环境，基于 Ollama + Python 构建，让开发者可以在本地运行 AI 辅助编码而无需依赖云服务。

## 核心功能

- **本地 LLM 支持** - 基于 Ollama 运行本地大模型
- **Python 集成** - 纯 Python 实现，易于集成
- **免费开源** - 无需付费即可使用
- **隐私友好** - 代码不上传云端

## 技术特点

- Python
- Ollama 后端
- 本地部署

## 应用场景

- 本地 AI 辅助编程
- 隐私敏感的代码工作
- 离线开发环境
- 学习与实验
