# agency-agents 🎭

## 1. 项目概述

### 1.1 项目名称
agency-agents (The Agency)

### 1.2 一句话概述
专门化的 AI agent 角色集合，每个 agent 具有独特个性和专业技能，可与 Claude Code 集成使用。

### 1.3 详细描述
The Agency 是一个精心设计的 AI agent 个性化集合。每个 agent 都是具有独特声音、沟通风格和专业领域的专家。与通用的 prompt 模板不同，这些 agent 是经过实战测试的工作流程，具有可衡量的成果。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: Agent 个性化设计
- **描述**: 每个 agent 都有独特的个性和专业领域
- **为什么重要**: 相比通用 prompt，专门的 agent 能提供更好的结果

#### 创新点 2: 开箱即用的 workflow
- **描述**: 包含经过测试的工作流程和成功指标
- **为什么重要**: 减少集成时间，直接可用

#### 创新点 3: Claude Code 深度集成
- **描述**: 可直接复制到 Claude Code agents 目录使用
- **为什么重要**: 无缝集成到现有开发流程

### 3.2 技术优势

| 优势类别 | 具体优势 | 量化指标 |
|---------|---------|---------|
| 专业性 | 多领域专门的 agent | 5+ 角色 |
| 可用性 | Claude Code 即插即用 | 一行命令集成 |
| 生产就绪 | 实战测试的工作流 | 可衡量成果 |

### 3.3 应用场景

1. **前端开发**: 使用专门的代码开发 agent
2. **社区管理**: Reddit 社区运营 agent
3. **内容创作**: 创意写作和内容生成
4. **质量保证**: 代码审查和测试
