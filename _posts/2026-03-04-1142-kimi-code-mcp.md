# Kimi-Code-MCP

## 1. 项目概述

### 1.1 项目名称
Kimi-Code-MCP

### 1.2 一句话概述
将Claude Code委托给Kimi K2.5进行大规模代码库分析，节省90%令牌成本

### 1.3 详细描述
Kimi-Code-MCP是一个MCP服务器，实现了Claude Code与Kimi K2.5（256K上下文）的集成。用户可以将大量代码库分析工作委托给Kimi，通过会话缓存和并行代理节省高达90%的令牌成本。支持TypeScript开发。
