# mcp-for-beginners - Microsoft MCP 入门教程

## 1. 项目概述

### 1.1 项目名称
mcp-for-beginners

### 1.2 一句话概述
Microsoft 官方推出的 Model Context Protocol (MCP) 开源入门课程，通过跨语言实际示例帮助开发者学习构建模块化、可扩展的 AI 工作流。

Microsoft 官方推出的 MCP (Model Context Protocol) 开源入门课程，涵盖 .NET, Java, TypeScript, JavaScript, Rust 和 Python 等多种语言的跨语言示例。课程专注于构建模块化、可扩展和安全的 AI 工作流技术。
