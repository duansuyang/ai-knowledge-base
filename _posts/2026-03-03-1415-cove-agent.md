# CoVe: Training Interactive Tool-Use Agents via Constraint-Guided Verification

## 1. 项目概述

### 1.1 项目名称
CoVe (Constraint-Verification)

### 1.2 一句话概述
新型后训练数据合成框架，通过约束验证训练交互式工具使用智能体，4B模型性能匹敌17倍大模型。

### 1.3 详细描述
CoVe是一个后训练数据合成框架，专门用于训练交互式工具使用智能体。框架首先定义显式任务约束，这些约束既指导复杂轨迹生成，又作为确定性验证器评估轨迹质量。基于此创建高质量训练轨迹用于监督微调(SFT)和强化学习(RL)。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 约束定义与验证
- **描述**: 定义显式任务约束，既指导轨迹生成又作为验证器
- **为什么重要**: 确保训练数据复杂度和正确性
- **对比现有方法**: 传统方法难以同时保证数据质量和复杂度

#### 创新点 2: 数据合成框架
- **描述**: 为SFT和RL生成高质量训练轨迹和奖励信号
- **为什么重要**: 解决交互式工具使用智能体的训练数据稀缺问题
- **对比现有方法**: 手动标注数据成本高且质量不稳定

### 3.2 技术优势

## 4. 技术细节

### 4.1 核心方法

#### Constraint-Verification框架
1. 定义显式任务约束
2. 约束指导生成复杂轨迹
3. 约束作为确定性验证器
4. 生成SFT训练轨迹
5. 导出RL奖励信号

### 4.2 模型
- CoVe-4B: 4B参数模型
- 在τ²-bench基准测试中表现优异
