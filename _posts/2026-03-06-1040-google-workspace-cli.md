# Google Workspace CLI

## 项目概述

Google官方出品的命令行工具,一个命令行工具整合Drive、Gmail、Calendar、Sheets、Docs、Chat、Admin等Google Workspace服务。动态从Google Discovery Service构建,包含AI agent skills。

## 核心特性

- **多服务整合**: Drive、Gmail、Calendar、Sheets、Docs、Chat、Admin
- **AI Agent集成**: 包含AI agent skills,可用于自动化任务
- **动态构建**: 基于Google Discovery Service自动生成
- **OAuth2认证**: 完整的认证流程支持
- **Rust开发**: 高性能、安全

## 技术栈

- 语言: Rust
- API: Google Discovery Service
- 认证: OAuth2
