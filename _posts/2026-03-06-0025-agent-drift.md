# agent-drift

## 1. 项目概述

### 1.1 项目名称
agent-drift

### 1.2 一句话概述
压力测试AI智能体的目标偏移和系统提示词违规的测试框架。

### 1.3 详细描述
agent-drift是一个用于压力测试AI智能体的工具，专注于检测目标偏移(goal drift)和系统提示词违规问题。该项目受ICLR 2026论文"Asymmetric Goal Drift in Coding Agents"启发。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 目标偏移检测
- **描述**: 自动检测智能体在执行任务过程中是否偏离原始目标
- **为什么重要**: 目标偏移是AI智能体面临的关键安全问题之一
- **对比现有方法**: 基于ICLR 2026最新研究，提供系统性检测方案

#### 创新点 2: 系统提示词违规检测
- **描述**: 监控智能体是否违反系统提示词中设定的约束
- **为什么重要**: 防止智能体绕过安全限制
- **对比现有方法**: 提供了红队测试能力

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 创新性 | 首个专注于目标偏移的开源测试框架 |
| 实用性 | 可集成到CI/CD流程中 |
| 适用场景 | AI安全研究、红队测试 |

### 3.3 应用场景

1. **AI安全研究**: 研究目标偏移的成因和解决方案
2. **红队测试**: 对AI智能体进行安全评估
