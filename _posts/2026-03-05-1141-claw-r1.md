# Claw-R1 - Agentic RL for Modern Agents

## 1. 项目概述

### 1.1 项目名称
Claw-R1

### 1.2 一句话概述
为OpenClaw等通用Agent打造的Agentic RL训练框架，引入中间件层连接Agent端和训练端。

### 1.3 详细描述
Claw-R1是一个训练框架，旨在弥合Agentic RL与通用Agent（如OpenClaw、Claude Code等）之间的关键差距。框架引入了中间件层（Gateway Server + DataPool）作为Agent端和训练端之间的唯一桥梁，支持三种模式：白盒离线、黑盒离线、黑盒在线服务。

## 3. 项目优势

### 3.1 核心创新点

**通用Agent的Agentic RL**: 传统Agentic RL框架通常依赖ReAct等简单代理，而通用代理（如OpenClaw、Claude Code、Open Code）提供了更丰富的能力——但现有的RL管道并未为此设计。

**通用Agent的Agentic RL**: 现代基础模型尚未完全适应在通用Agent架构中发挥更大作用。Claw-R1使模型能够在这些下一代代理中发挥更大、更有效的作用。

**中间件层设计**: 引入Gateway Server + DataPool作为唯一桥梁，支持三种训练模式。

### 3.2 技术优势

| 优势类别 | 具体优势 | 说明 |
|---------|---------|-----|
| 异步训练 | 异步训练与Rollout | 解耦RL训练与 rollout |
| Agent-训练解耦 | 支持在线服务 | 无需数据集 |
| 零代码侵入 | 只需配置base_url | LangChain, AutoGen, CrewAI等即插即用 |

### 3.3 应用场景

1. **通用Agent后训练**: 为OpenClaw等通用Agent进行RL后训练
2. **在线服务优化**: 对接实时用户请求持续优化模型
3. **多框架集成**: 支持LangChain、AutoGen、CrewAI等框架

```
┌─────────────────────────────────────────────────────┐
│                  Training Side                       │
│  ┌─────────────┐                                    │

└─────────────────────────────────────────────────────┘
                         ▲

┌─────────────────────────────────────────────────────┐
│               Middleware Layer                       │
│  ┌─────────────┐    ┌─────────────┐               │
│  │   Gateway   │    │  DataPool   │               │
│  │   Server    │    │             │               │

└─────────────────────────────────────────────────────┘
                         ▲
                         │ HTTP
                         ▼
```

### 4.2 支持的模式

1. **白盒离线**: 可访问Agent内部状态
2. **黑盒离线**: 通过API调用Agent
3. **黑盒在线**: 实时服务模式，数据流从用户请求到DataPool

### 5.2 综合评价

Claw-R1填补了Agentic RL领域的一个重要空白。它是第一个专门为通用Agent（如OpenClaw）设计的RL训练框架。中间件层的创新设计使得可以灵活支持各种训练模式，特别是对黑盒Agent的支持使得集成现有系统变得非常简单。

1. 首个专门为通用Agent设计的RL框架

4. 零代码侵入现有Agent系统
