# AlphaSearch - MCTS Document Retrieval

## 1. 项目概述

### 1.1 项目名称
AlphaSearch

### 1.2 一句话概述
AlphaGo风格的MCTS文档检索，用LLM推理替代传统向量嵌入。

### 1.3 详细描述
AlphaSearch用蒙特卡洛树搜索(MCTS)替代向量数据库进行文档检索。只需上传PDF、提问，系统用LLM推理找到包含答案的精确段落并给出引用。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 无向量检索
- **描述**: 完全摒弃向量数据库、嵌入、分块，用MCTS直接推理
- **为什么重要**: 传统RAG依赖chunking和嵌入，可能丢失文档结构信息
- **对比现有方法**: 传统方法用向量相似度，本文用MCTS做推理导航

#### 创新点 2: 分层MCTS搜索
- **描述**: Phase 0路由 → Phase 1文档选择 → Phase 2段落搜索 → Deep Read
- **为什么重要**: 分层搜索更精确，每层用不同粒度的summary做导航

#### 创新点 3: 视觉Deep Read
- **描述**: 用GPT-4o Vision读取原始PDF页面提取答案，而非仅依赖summary
- **为什么重要**: 保证答案准确性，引用原文

## 4. 技术细节

### 4.1 核心方法

#### MCTS检索流程
```
User Query → Phase 0 (路由) → Phase 1 (选文档) → Phase 2 (找段落) → Deep Read → Answer
```

- **导航阶段**: 用GPT-4o-mini生成summary做MCTS导航（便宜）
- **答案阶段**: 用GPT-4o Vision读取原始PDF页面提取答案（准确）
