# Utonia: Toward One Encoder for All Point Clouds

## 1. Project Overview

### 1.1 Project Name
Utonia

### 1.2 One-sentence Summary
A unified self-supervised point transformer encoder that works across all point cloud domains - remote sensing, outdoor LiDAR, indoor RGB-D, CAD models, and video-lifted point clouds.

### 1.3 Detailed Description
Utonia is the first attempt to train a single self-supervised point transformer encoder across diverse domains. Despite distinct sensing geometries, densities, and priors, Utonia learns a consistent representation space that transfers across domains. This unification improves perception capability while revealing emergent behaviors when domains are trained jointly. It also benefits embodied and multimodal reasoning tasks.

## 4. Technical Details

### 4.1 Core Method

#### Method Description
Utonia uses a self-supervised transformer encoder trained on diverse point cloud domains including:
- Remote sensing point clouds
- Outdoor LiDAR scans
- Indoor RGB-D sequences
- Object-centric CAD models
- Point clouds lifted from RGB-only videos

The unified representation space allows knowledge transfer between domains.
