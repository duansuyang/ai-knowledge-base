# appwrite/arena - Appwrite SDK AI 模型基准测试

## 项目概述

**appwrite/arena** 是一个基准测试项目，用于评估 AI 模型对 Appwrite SDK 和服务的理解程度。测试 LLMs 在有/无 skill 文件的情况下对 Appwrite 的掌握情况。

## 核心亮点

- **SDK 理解评估**: 测试 AI 模型对 Appwrite API 的理解
- **Skill 文件对比**: 对比有/无 skill 的模型表现
- **开源基准**: 公开的评估框架

## 技术特点

- Appwrite SDK 集成
- LLM 能力评估
- Skill 文件系统

## 实验维度

1. 基础 API 调用能力
2. 错误处理能力
3. Skill 增强效果

## 应用场景

1. 评估 AI 模型的 API 理解能力
2. 改进 AI 编程助手的 SDK 支持
3. 开发更好的 skill 文件

## 适合人群

- AI 模型开发者
- SDK 维护者
- AI 编程助手用户
