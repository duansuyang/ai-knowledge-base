# RuView - WiFi DensePose

## 1. 项目概述

### 1.1 项目名称
RuView

### 1.2 一句话概述
利用WiFi信号实现实时人体姿态估计、生命体征监测和存在检测。

### 1.3 详细描述
RuView是一项利用商品WiFi信号实现实时人体姿态估计、生命体征监测和存在检测的技术。整个过程不需要任何视频像素，真正实现了隐私保护的感知系统。该项目已获得22,905颗GitHub星标和13,054颗本周星标。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 隐私保护的视觉感知
- **描述**: 不使用任何摄像头或视频，仅通过WiFi信号实现感知
- **为什么重要**: 解决隐私敏感场景下的监控需求

#### 创新点 2: 多功能感知
- **描述**: 同时支持姿态估计、生命体征监测和存在检测
- **为什么重要**: 一个系统满足多种感知需求

### 3.2 技术特点

| 特点 | 说明 |
|------|-----|
| 技术基础 | WiFi信号处理 (CSI) |
| 语言 | Rust |
| 硬件需求 | 商品WiFi设备 |
| 隐私 | 纯信号处理，无需摄像头 |

### 3.3 应用场景

1. **智能家居**: 无摄像头入侵检测
2. **医疗监护**: 非接触式生命体征监测
3. **老人护理**: 跌倒检测和活动监测
4. **隐私敏感环境**: 需要监控但不侵犯隐私的场景
