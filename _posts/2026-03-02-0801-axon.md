# Axon - Graph-Powered Code Intelligence

## 项目概述

Axon 是一个图驱动的代码智能引擎，将代码库索引为知识图谱，通过 MCP 工具暴露给 AI Agent，同时提供 CLI 供开发者使用。

## 核心功能

- **代码索引** - 将代码库结构转换为知识图谱
- **MCP 工具** - 为 Claude Code 等 AI Agent 提供代码理解工具
- **死代码检测** - 自动识别未使用的代码
- **Tree-sitter 支持** - 精确的代码解析

## 技术特点

- Python 实现
- 支持 TypeScript 代码分析
- MCP 协议集成
- 知识图谱构建

## 应用场景

- AI 代码辅助理解
- 代码库分析
- 死代码清理
- 代码搜索与导航
