# jina-cli

## 项目概述

轻量级CLI工具，封装Jina AI Reader API，为AI Agents提供URL获取和解析能力。支持将任意URL转换为LLM友好的格式(Markdown/Text/HTML)，特别适合读取X(Twitter)、博客和新闻网站。

## 核心功能

- **URL解析**: 一键将网页转换为Markdown格式
- **CLI友好**: 专为AI Agent设计，可集成到各种工作流
- **多格式输出**: 支持Markdown、Text、HTML
- **社交媒体支持**: 特别优化了Twitter/X的解析

## 使用场景

1. AI Agent获取网页内容
2. 新闻摘要自动化
3. 博客内容抓取
4. 竞争情报收集

## 安装使用

```bash
go install github.com/geekjourneyx/jina-cli@latest
jina-cli https://example.com
```

## 技术栈

- Go语言
- Jina AI Reader API
