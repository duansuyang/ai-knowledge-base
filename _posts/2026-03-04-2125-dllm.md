# dLLM - 扩散语言模型训练框架

## 1. 项目概述

### 1.1 项目名称
dLLM (Diffusion Language Modeling)

### 1.2 一句话概述
统一扩散语言模型训练和评估的开源库，支持 LLaDA、Dream 等主流扩散模型。

### 1.3 详细描述
dLLM 是一个用于扩散语言模型训练和评估的统一框架，提供了可扩展的训练管道（基于 HuggingFace Trainer），支持 LoRA、DeepSpeed、FSDP 等技术，以及统一的评估管道（基于 lm-evaluation-harness）。

- [x] 多模态 (生成式AI)

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 统一训练框架
- **描述**: 整合了多种扩散语言模型的训练流程，包括 MDLM (masked diffusion)、BD3LM (block diffusion)、Edit Flows 等
- **为什么重要**: 降低了扩散语言模型的训练门槛

#### 创新点 2: 支持主流扩散模型
- **描述**: 提供了 LLaDA、Dream 等开源模型的最小训练/推理/评估配方
- **为什么重要**: 开发者可以直接使用这些配方训练自己的模型

#### 创新点 3: 推理加速支持
- **描述**: 支持 Fast-dLLM 加速推理和评估，包括 cache 和 confidence-threshold decoding
- **为什么重要**: 解决了扩散模型推理慢的问题

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 训练支持 | LoRA, DeepSpeed, FSDP |
| 推理加速 | Fast-dLLM 支持 |
| 模型转换 | 可将任意 AR 模型转换为扩散模型 |

### 3.3 应用场景

1. **扩散语言模型研究**: 提供完整的研究工具链
2. **模型训练**: 使用提供的配方训练自己的扩散模型
3. **模型评估**: 统一的评估管道

## 4. 技术细节

### 4.1 核心方法

dLLM 基于 HuggingFace Transformers Trainer 和 lm-evaluation-h构建，提供：
- 可扩展的训练管道
- 统一的评估管道
- 最小训练/推理/评估配方

### 4.2 支持的模型和算法

- LLaDA
- Dream
- MDLM (masked diffusion)
- BD3LM (block diffusion)
- Edit Flows
- Fast-dLLM

### 5.2 综合评价

dLLM 是扩散语言模型领域的重要工具。它不仅提供了完整的训练和评估流程，还包含了多种主流扩散模型的实现。对于研究人员和开发者来说，这是一个非常有价值的资源，可以快速上手扩散语言模型的研究和开发。

### 5.3 值得关注的点

1. **Tiny-A2D**: 发布了 SOTA 小型扩散模型 (0.5B/0.6B)
2. **BERT-Chat**: BERT 聊天扩散模型
3. **模型转换**: 可将任意 AR 模型转换为扩散模型
