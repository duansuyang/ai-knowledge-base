# ClawShield - AI Agent安全代理

## 1. 项目概述

### 1.1 一句话概述
ClawShield是一个面向AI Agent的安全代理，可扫描每条消息检测提示词注入、PII和敏感信息，提供深度防御机制。

### 1.2 详细描述
ClawShield是由SleuthCo开发的AI Agent安全代理框架，采用Go语言实现，提供Defense-in-depth（深度防御）安全架构。核心功能包括：提示词注入检测、个人身份信息(PII)检测、敏感密钥扫描。技术架构包含Go反向代理 + iptables防火墙 + eBPF内核监控，支持YAML策略引擎和审计日志，并内置5个基于RAG知识库的AI Agent。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 深度防御架构
- **描述**: Go反向代理 + iptables防火墙 + eBPF内核监控的三层防护
- **为什么重要**: 从网络层到应用层提供全方位保护，检测多层次攻击
- **对比现有方法**: 传统Agent安全方案通常只做单一层面检测

#### 创新点 2: 多维度安全检测
- **描述**: 同时检测提示词注入、PII泄露、敏感密钥泄露
- **为什么重要**: 覆盖AI Agent最常见的安全风险
- **对比现有方法**: 大多数方案只关注单一安全维度

#### 创新点 3: RAG增强的知识库
- **描述**: 内置5个AI Agent，每个都有基于RAG的安全知识库
- **为什么重要**: 可实现上下文感知的智能安全决策
- **对比现有方法**: 传统规则引擎缺乏语义理解能力

### 3.2 功能特性

| 功能 | 说明 |
|------|-----|
| 提示词注入检测 | 识别恶意提示词注入攻击 |
| PII检测 | 自动检测和脱敏个人身份信息 |
| 敏感密钥扫描 | 检测API密钥、密码等敏感凭证 |
| 防火墙集成 | iptables联动阻断恶意流量 |
| eBPF监控 | 内核级行为监控 |
| YAML策略引擎 | 可定制的安全策略 |
| 审计日志 | 完整的安全事件记录 |

### 3.3 应用场景

1. **企业AI部署**: 为LLM API提供安全网关
2. **Agent平台安全**: 保护Multi-Agent系统通信安全
3. **数据合规**: PII检测满足GDPR等法规要求
4. **API安全**: 防止敏感密钥泄露
