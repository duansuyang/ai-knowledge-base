# awesome-llm-apps

## 1. 项目概述

### 1.1 项目名称
awesome-llm-apps

### 1.2 一句话概述
精选的LLM应用集合，包含RAG、AI智能体、多智能体团队、MCP、语音代理等实现。

### 1.3 详细描述
一个精心策划的LLM应用集合，展示了使用RAG、AI Agents、Multi-agent Teams、MCP、Voice Agents等技术构建的应用。该仓库收录了使用OpenAI、Anthropic、Google、xAI及开源模型（如Qwen、Llama）的LLM应用，这些模型可以在本地计算机上运行。

### 1.4 所属领域
- [x] Agent/IAG (智能体、工作流、MCP、Tool Use)

- [x] 多模态 (VLM、视频生成、图像生成)
- [ ] 强化学习 (RLHF、GRPO、DPO)

- [x] RAG/知识库 (向量检索、Hybrid Search)
- [ ] AI安全 (对齐、护栏)
- [ ] 训练/部署 (推理引擎、分布式训练)
