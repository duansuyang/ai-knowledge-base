# OpenClaw PG Memory

## 项目概述

OpenClaw PG Memory 是为 OpenClaw agents 提供的持久化内存解决方案，基于 PostgreSQL + pgvector + Ollama 构建，提供向量搜索、自动备份、Web Dashboard 和 CLI 工具。

## 核心功能

- **向量搜索**: 基于 pgvector 的语义搜索能力
- **自动备份**: 数据安全保障
- **Web Dashboard**: 可视化管理界面
- **CLI 工具**: 便捷命令行操作
- **5分钟快速部署**: 简化安装流程

## 技术栈

- PostgreSQL + pgvector
- Ollama
- Python

## 安装使用

```bash
# 快速安装
pip install openclaw-pg-memory
```
