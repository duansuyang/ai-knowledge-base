# 🔬 recall - AI 智能体记忆技能

## 一句话概述

让 Claude Code 和 Codex 能够回忆过去对话的技能插件，解决 AI 智能体无法跨会话记忆的问题。

## 项目描述

recall 是一个为 Claude Code 和 Codex 设计的技能(Skill)，它赋予 AI 智能体记住过去对话的能力。当 AI 编码助手需要回顾之前的项目上下文时，这个技能可以让 agent 自动检索和记忆之前的对话内容。

## 所属领域

- [x] Agent/IAG (智能体、工作流、MCP、Tool Use)

## 核心亮点

- **跨会话记忆**: 让 AI agent 能够记住之前的对话
- **即插即用**: 作为 Skill 形式添加，无需复杂配置
- **兼容主流工具**: 支持 Claude Code 和 Codex

## 技术细节

该技能通过将对话历史存储在本地，并在需要时检索来实现记忆功能。具体的实现方式可能涉及：
- 本地存储对话历史
- 语义检索相关上下文
- 自动注入到新对话中

## 应用场景

1. **项目连续性**: 当用户重新打开一个项目时，AI 能记住之前的讨论和决策
2. **多任务处理**: 在处理多个相关任务时保持上下文一致
3. **长期项目**: 在长时间开发周期中维护项目知识
