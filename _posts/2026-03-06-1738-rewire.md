# Rewire - AI 工作助手设计 Agent

## 项目概述

Rewire 是一个帮你设计专属 Claude Code Agent 的 AI。它不给你技术方案，而是给你讲一个"故事"——让你在脑子里"看见"AI 是怎么帮你工作的。

## 核心功能

1. **调研工作场景** - 追问细节，亲手试用工具，搜索开源方案
2. **输出完整方案** - 不是技术报告，而是可直接使用的 Claude Code 项目
3. **包含指令文件 + 工具接入 + 启动验证**

## 工作流程

```
描述工作 → Agent 追问细节 → 亲手试用工具 → 搜索开源方案 → 输出可运行项目
```

## 示例场景

- 运营：用户反馈分类（2-3小时 → 5分钟审核）
- 自媒体：选题调研（3-4小时刷平台 → 10分钟清单）
- HR招聘：简历筛选（24小时自动筛选）

## 底层哲学

- 工作闭环：调研 → 决策 → 规划 → 执行 → 反馈 → 复盘
- 人应该从"管道"变成"阀门"（只做判断、兜异常）
- 最先自动化的是让你难受的那步，而非最耗时的
