# soul.py

## 1. 项目概述

### 1.1 项目名称
soul.py

### 1.2 一句话概述
为任意 LLM 智能体提供持久化身份和记忆的 Markdown 本地存储方案

### 1.3 详细描述
一个与模型无关的智能体记忆管理框架，支持 Markdown 格式的持久化存储，帮助 LLM 智能体保持跨会话的身份一致性和记忆连续性。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 跨会话记忆持久化
- **描述**: 使用 Markdown 文件本地存储智能体记忆
- **为什么重要**: 解决 LLM 智能体"每次会话都是新开始"的问题
- **对比现有方法**: 比向量数据库更简单，比纯内存更持久

#### 创新点 2: Provider-Agnostic 设计
- **描述**: 不依赖特定 LLM 提供商
- **为什么重要**: 可适配任意 LLM API
- **对比现有方法**: 大多数记忆方案绑定特定框架

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 易用性 | 纯 Python，无复杂依赖 |
| 可移植性 | Markdown 格式便于阅读和迁移 |
| 通用性 | 支持任意 LLM 提供商 |

### 3.3 应用场景

1. **AI 助手**: 保持个性化记忆和身份
2. **对话系统**: 记住用户偏好和历史交互
3. **智能体**: 构建长期记忆能力
