# OpenRAG-Skill - Prompt-Only RAG with Strict Citations

## 1. 项目概述

### 1.1 项目名称
OpenRAG-Skill

### 1.2 一句话概述
面向上下文文档的纯 Prompt 引用型 RAG Skill，支持严格的引用溯源。

### 1.3 详细描述
OpenRAG-Skill 是一个专注于引用准确性的 RAG（检索增强生成）工具。它通过纯 Prompt 的方式实现 RAG 功能，特别强调生成内容必须严格引用原始文档，适合需要高准确性和可追溯性的应用场景。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 严格引用机制
- **描述**: 确保生成的每个事实都有对应的文档引用
- **为什么重要**: 解决 RAG 幻觉问题，提高可信度
- **对比现有方法**: 传统 RAG 可能产生无引用的事实，OpenRAG 强制引用

#### 创新点 2: 纯 Prompt 实现
- **描述**: 无需复杂的后端架构，通过 Prompt 工程实现 RAG
- **为什么重要**: 降低部署门槛，易于集成
- **对比现有方法**: 需要向量数据库+复杂后端，纯 Prompt 更轻量

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 准确性 | 严格引用减少幻觉 |
| 易用性 | 纯 Prompt 实现，部署简单 |
| 可追溯性 | 每句话都有来源 |

### 3.3 应用场景

1. **法律/合规文档问答**: 需要严格溯源的领域
2. **学术文献助手**: 引用必须准确
3. **企业内部知识库**: 敏感信息需要明确来源
