# AgentSkillOS: Agent技能生态系统管理框架

## 1. 项目概述

### 1.1 项目名称
AgentSkillOS: Organizing, Orchestrating, and Benchmarking Agent Skills at Ecosystem Scale

### 1.2 一句话概述
首个针对Agent技能选择、编排和生态系统级管理的原则性框架。

### 1.3 详细描述
随着Claude Agent技能的快速普及，如何有效利用、管理和扩展Agent技能生态系统成为核心问题。AgentSkillOS提出两阶段框架：(i) Manage Skills - 通过节点级递归分类将技能组织成能力树；(ii) Solve Tasks - 通过DAG管道检索、编排和执行多个技能。

### 1.4 所属领域
- [x] Agent/IAG (智能体、工作流、MCP、Tool Use)
- [ ] LLM/大模型 (预训练、微调、推理、量化)
- [ ] 多模态 (VLM、视频生成、图像生成)
- [ ] 强化学习 (RLHF、GRPO、DPO)
- [ ] 具身智能 (机器人、Sim2Real)
- [ ] RAG/知识库 (向量检索、Hybrid Search)
- [ ] AI安全 (对齐、护栏)
- [ ] 训练/部署 (推理引擎、分布式训练)
