# Star-Office-UI - 像素风格多代理协作看板

## 1. 项目概述

### 1.1 一句话概述
一个将 AI 代理工作状态可视化的像素风格办公室看板，让团队直观看到"谁在做什么"。

### 1.2 详细描述
Star Office UI 是一个"多人协作状态看板"——AI 助手会根据状态自动走到不同位置（休息区/工作区/bug区），还能看到他们昨天的工作小记。支持多 Agent 机制，可邀请访客加入办公室。

## 3. 项目优势

### 3.1 核心创新点

**创新点 1: 可视化 Agent 工作状态**
- 将抽象的 AI 工作状态映射到具体的物理位置
- 状态包括: idle/writing/researching/executing/syncing/error
- 通过像素动画和气泡展示实时状态

**创新点 2: 多 Agent 协作机制**
- 支持邀请其他 Agent 加入办公室
- Agent 可 push 自己的状态到看板
- 支持 join key 认证

**创新点 3: 昨日小记功能**
- 自动从 memory/*.md 读取昨日工作记录
- 展示在办公室看板上
- 基础脱敏处理

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 易用性 | 一键启动，支持手机端访问 |
| 可扩展性 | 支持多 Agent 接入 |
| 创新性 | 独特的像素风格可视化 |

### 3.3 应用场景

1. **团队协作监控**: 实时查看多个 AI 代理的工作状态
2. **工作日志回顾**: 了解代理昨天完成了什么任务
3. **多代理协调**: 让多个 AI 代理协同工作
