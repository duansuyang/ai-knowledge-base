# kibitz - AI Agent 实时监控工具

## 1. 项目概述

### 1.1 一句话概述
kibitz 是一个实时解码 AI Agent 行为的工具，可监控多个 Claude Code 和 Codex 会话。

### 1.2 详细描述
kibitz 提供实时解码的 AI Agent 行为反馈，让开发者可以监控多个 Claude Code 和 Codex 会话，查看每个 Agent 正在做什么，并高效协调 swarm。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 实时 Agent 监控
- **描述**: 提供实时解码的 Agent 行为流，开发者可以看到 Agent 的每一个动作
- **为什么重要**: 解决 Agent "黑盒"问题，提高可观测性

#### 创新点 2: 多会话管理
- **描述**: 支持同时监控多个 Claude Code 和 Codex 会话
- **为什么重要**: 方便管理和协调多个 Agent

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 可观测性 | 实时查看 Agent 行为 |
| 多会话 | 同时监控多个 Agent |
| 协调能力 | 支持 swarm 协调 |

### 3.3 应用场景

1. **开发调试**: 实时查看 Agent 决策过程
2. **多 Agent 协调**: 管理多个 Agent 会话
3. **团队协作**: 团队成员共享 Agent 状态
