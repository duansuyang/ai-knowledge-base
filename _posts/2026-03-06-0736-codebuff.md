# Codebuff - Open Source AI Coding Assistant

## 项目概述

Codebuff is an open-source AI coding assistant that edits your codebase through natural language instructions. Unlike single-model tools, it coordinates specialized agents that work together to understand your project and make precise changes.

## 核心特性

1. **Multi-Agent架构**: 协调多个专业Agent（文件选择器、规划器、编辑器、审查器）
2. **CLI工具**: 通过 `npm install -g codebuff` 安装
3. **SDK支持**: 可集成到生产环境
4. **支持任意OpenRouter模型**: 不局限于Anthropic模型

## 性能表现

- 在175+编程任务上超越Claude Code: **61% vs 53%**
- 6000+ commits，持续活跃开发

## 工作流程

当用户要求"add authentication to my API"时:
1. File Picker Agent: 扫描代码库理解架构
2. Planner Agent: 规划需要修改的文件
3. Editor Agent: 精确修改代码
4. Reviewer Agent: 验证修改正确性

## 技术栈

- TypeScript 97.2%
- 开源协议: Apache-2.0

## 亮点

- 多Agent协作，更好的上下文理解
- 更精准的代码修改，更少错误
- 可自定义Agent和工作流
- 支持任何OpenRouter模型
