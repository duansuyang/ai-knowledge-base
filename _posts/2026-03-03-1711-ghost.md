# ghost - Git is the coding agent

## 项目概述

ghost 是一个以 Git 为核心的 AI 编程代理。它将 Git 版本控制能力与 AI 代理相结合，让开发者可以通过自然语言指令来完成代码管理任务。

## 核心特性

- **Git 原生集成**：直接利用 Git 的版本控制能力
- **AI 驱动**：通过 AI 理解开发者意图
- **CLI 优先**：通过命令行界面操作
- **Shell 实现**：使用 Shell 脚本构建

## 技术栈

- Shell
- Git
- AI/LLM 集成

## 为什么值得关注

1. **创新性**：将版本控制与 AI 代理结合，是开发工作流的新尝试
2. **实用性**：直接面向开发者日常工作场景
3. **轻量级**：基于 Shell 实现，易于部署和使用

## 应用场景

- 自动化代码提交和分支管理
- 代码 Code Review 辅助
- 版本历史分析和查询
- 自动化发布流程

## 适合人群

- 开发者
- DevOps 工程师
- 技术团队
