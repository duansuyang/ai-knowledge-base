# seomachine - SEO内容创作AI工具

## 1. 项目概述

### 1.1 项目名称
seomachine

### 1.2 一句话概述
使用Claude Code构建的专业SEO优化长文本博客内容创作工作区

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 多代理协作系统
- **描述**: 内置10+专业代理，包括内容分析、SEO优化、元标签创建、内部链接、关键词映射、编辑器、性能分析、标题生成器、CRO分析师
- **为什么重要**: 自动化SEO内容创作全流程

#### 创新点 2: 数据源集成
- **描述**: 支持Google Analytics 4、Google Search Console、DataForSEO实时数据集成
- **为什么重要**: 基于真实数据制定内容策略

#### 创新点 3: 26个营销技能
- **描述**: 内置完整的营销技能库，涵盖文案、CRO、A/B测试、邮件序列、定价策略等
- **为什么重要**: 提供全方位营销支持

### 3.2 技术优势

| 优势类别 | 具体优势 | 量化指标 |
|---------|---------|---------|
| 自动化 | 全流程SEO内容创作 | 研究→写作→优化→发布 |
| 集成度 | WordPress直接发布 | REST API + Yoast SEO |
| 分析能力 | 5大Python分析模块 | 搜索意图、关键词、SEO质量、内容长度、可读性 |

### 3.3 应用场景

1. **内容营销**: 自动化创建SEO优化的博客文章

## 4. 技术细节

### 4.1 核心方法/架构

基于Claude Code的自定义命令和工作区配置，包含：
- `.claude/commands/` - 自定义工作流命令
- `.claude/agents/` - 专业分析代理
- `.claude/skills/` - 26个营销技能
- `data_sources/modules/` - Python分析模块

### 4.2 技术栈

| 类别 | 技术/工具 |
|------|----------|
| 框架 | Claude Code |
| 库 | Google Analytics API, DataForSEO API, nltk, scikit-learn |
| 部署 | WordPress REST API |

## 5. 实验流程与结果

### 5.1 功能特性

- `/research [topic]` - 关键词和竞品研究
- `/write [topic]` - 创建2000-3000+词SEO文章
- `/rewrite [topic]` - 更新现有内容
- `/analyze-existing [URL]` - 分析现有文章
- `/optimize [file]` - 最终SEO优化
- `/publish-draft [file]` - 发布到WordPress
