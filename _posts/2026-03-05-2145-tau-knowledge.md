# τ-Knowledge - Agent RAG Benchmark

## 1. 项目概述

### 1.1 项目名称
τ-Knowledge (Tau-Knowledge)

### 1.2 一句话概述
评估Agent在非结构化知识库中进行长程交互的benchmark，覆盖金融客服场景。

### 1.3 详细描述
对话Agent越来越多地部署在知识密集型场景中，但现有benchmark独立评估检索或工具使用，未能覆盖真实的长程交互。τ-Knowledge扩展τ-Bench，评估Agent协调外部自然语言知识与工具输出的能力。

### 1.4 所属领域
- [x] Agent/IAG
- [x] RAG/知识库
- [ ] LLM/大模型
- [ ] 多模态
- [ ] 其他

## 2. 核心创新

### 2.1 τ-Banking 领域
- 模拟真实金融客服工作流
- 700+互联知识文档
- 工具执行的账户更新

### 2.2 评估结果
- 即使前沿模型仅达到 ~25.5% pass@1
- 可靠性随重复试验急剧下降
- Agent难以从密集关联知识库中检索正确文档
