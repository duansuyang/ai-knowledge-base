# aso-skills: AI Agent Skills for App Store Optimization

## Project Overview

**aso-skills** is a curated collection of AI agent skills specifically designed for App Store Optimization (ASO) and mobile app marketing. Built for indie developers, app marketers, and growth teams who want AI assistants like Cursor, Claude Code, or any Agent Skills-compatible AI to help with:

- Keyword research
- Metadata optimization
- Competitor analysis
- App growth strategies

## Key Features

- **Comprehensive ASO Skills**: Pre-built prompts and workflows for common ASO tasks
- **Multi-Platform Support**: Works with various AI coding agents
- **Marketing Automation**: Automates repetitive ASO workflows
- **Competitor Insights**: Tools for analyzing competing apps

## Use Cases

1. **Keyword Research**: Identify high-impact keywords for app visibility
2. **Metadata Optimization**: Optimize app title, description, and keywords
3. **Competitor Analysis**: Analyze competitor app strategies
4. **Growth Tracking**: Monitor and improve app store rankings
