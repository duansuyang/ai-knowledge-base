# Backstage - 像素艺术办公室中的 AI 智能体可视化

## 1. 项目概述

### 1.1 项目名称
Claude Backstage

### 1.2 一句话概述
让 AI 智能体在像素艺术办公室中可视化工作过程的 Claude Code 插件。

### 1.3 详细描述
Backstage 是一个创新的 Claude Code 插件，将终端 AI 工作流程转变为生动的像素艺术创业办公室。当 Claude 将工作委托给子智能体时，每个子智能体变成一个角色——他们走到工位、坐下、编码，完成后前往休息室，全程实时显示在像素艺术世界中。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 智能体可视化
- **描述**: 将 AI 智能体的工作过程可视化为像素艺术角色在办公室中移动
- **为什么重要**: 让开发者直观地观察多智能体系统的工作状态
- **对比现有方法**: 传统方式只能通过日志查看，缺少视觉反馈

#### 创新点 2: 子智能体追踪
- **描述**: 每个子智能体拥有独立角色，在办公室中有独立工位
- **为什么重要**: 便于理解和调试复杂的多智能体工作流
- **创新性**: 业界首个将游戏化元素引入 AI 智能体调试的工具

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 可视化 | 实时显示智能体状态和位置 |
| 交互性 | 可与角色互动，按空格键查看智能体状态 |
| 轻量级 | 基于 Bun 的轻量级服务器 |
| 可扩展 | 支持自定义角色和行为 |

### 3.3 应用场景

1. **多智能体开发调试**: 可视化观察多个子智能体的工作分配和执行状态
2. **智能体教学**: 通过游戏化方式理解智能体工作原理
3. **工作流演示**: 向非技术人员展示 AI 工作流程

## 4. 技术细节

### 4.1 核心方法/架构

#### 4.1.1 技术栈
- **运行时**: Bun
- **插件系统**: Claude Code Plugin
- **前端**: Pixel Art (Gather Town 风格)

#### 4.1.2 工作流程
1. Claude Code 启动插件服务器
2. 浏览器打开 http://localhost:7777
3. 子智能体被创建为像素角色
4. 角色根据任务状态在办公室中移动

### 4.2 安装配置

```bash
# 安装插件
claude plugin marketplace add https://github.com/sebyul2/backstage.git
claude plugin install backstage

# 启动查看器
/claude:server on
```
