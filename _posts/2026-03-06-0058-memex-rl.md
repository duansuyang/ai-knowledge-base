# Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

## 1. 项目概述

### 1.1 项目名称
Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

### 1.2 一句话概述
通过外部索引记忆机制解决LLM智能体在长程任务中的上下文窗口瓶颈，实现无损记忆压缩。

### 1.3 详细描述
LLM智能体在长程任务中面临有限上下文窗口的根本瓶颈。现有方案通过截断或摘要压缩上下文，但这些方法本质上是有损的，因为它们压缩或丢弃了过去的证据。Memex引入了一种索引体验记忆机制，在不丢弃证据的情况下压缩上下文。

## 4. 技术细节

### 4.1 核心方法/架构

#### 4.1.1 方法名称
Memex: Indexed Experience Memory

#### 4.1.2 方法描述
Memex维护一个紧凑的工作上下文，包含简洁的结构化摘要和稳定的索引，同时将完整保真度的底层交互存储在外部体验数据库中。智能体可以决定何时取消引用索引并恢复当前子目标所需的具体过去证据。通过MemexRL框架优化读写行为。

##### 方法流程图
```
Step 1: 接收长程任务输入
Step 2: 生成结构化摘要 + 创建索引
Step 3: 将完整交互存储到外部数据库
Step 4: 任务需要时检索相关索引
Step 5: 恢复精确过去证据
```

## 5. 实验流程与结果

### 5.1 实验设置

#### 5.1.1 数据集
长程挑战任务 benchmark

#### 5.1.2 评估指标

### 5.2 主实验结果

在挑战性长程任务上，使用MemexRL训练的Memex智能体在显著更小的工作上下文下提高了任务成功率。

### 5.3 局限性分析

#### 5.3.1 已知的局限性

1. **外部存储依赖**: 需要维护外部体验数据库
2. **检索延迟**: 索引检索可能引入额外延迟
