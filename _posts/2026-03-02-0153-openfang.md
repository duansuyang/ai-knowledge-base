# openfang - 开源 Agent 操作系统

## 1. 项目概述

### 1.1 一句话概述
OpenFang 是一个开源的 Agent 操作系统 (Agent OS)，为 AI Agent 提供完整的运行环境和基础设施。

### 1.2 详细描述
OpenFang 旨在构建一个完整的 Agent 生态系统，提供Agent运行所需的基础设施、工具集成和编排能力。支持 MCP 协议，可与各种 LLM 无缝集成。
