# Shannon - AI Autonomous Pentester

## 项目概述

Shannon 是 Keygraph 开发的自主式 AI 白盒渗透测试工具，专门用于 Web 应用程序和 API 的安全测试。它通过结合源代码分析和动态漏洞利用，自动识别攻击向量并执行真实漏洞利用。

## 核心特性

### 🎯 主要功能
- **全自动化操作**: 单个命令即可启动完整渗透测试，支持 2FA/TOTP 登录
- **可复现 PoC**: 最终报告仅包含经过验证的漏洞，带有可复现的利用代码
- **OWASP 覆盖**: 覆盖注入、XSS、SSRF、认证授权等漏洞类型
- **代码感知**: 分析源代码引导攻击策略，再通过动态利用验证
- **并行处理**: 漏洞分析和利用阶段并行运行

### 📊 性能表现
- **XBOW Benchmark**: 96.15% 成功率 (100/104 exploits)
- **OWASP Juice Shop**: 单次运行发现 20+ 漏洞
- **典型运行时间**: 1-1.5 小时
- **预估成本**: 约 $50 USD (使用 Claude 4.5 Sonnet)

### 🏗️ 架构设计
```
Reconnaissance → Vuln Analysis (并行) → Exploitation (并行) → Reporting
```

四阶段多智能体架构:
1. **侦察阶段**: 分析源代码，集成 Nmap、Subfinder、WhatWeb
2. **漏洞分析**: 并行检测各类漏洞
3. **漏洞利用**: 执行真实攻击验证
4. **报告生成**: 输出专业级渗透测试报告

## 产品线

| 版本 | 许可 | 用途 |
|------|------|------|
| Shannon Lite | AGPL-3.0 | 开源版，核心自动渗透测试框架 |
| Shannon Pro | 商业版 | 完整 AppSec 平台 (SAST + SCA + 商业逻辑测试 + 渗透测试) |

## 技术亮点

1. **静态-动态关联**: Pro 版将静态分析发现的漏洞通过动态利用验证
2. **代码属性图 (CPG)**: Pro 版使用 CPG 进行数据流分析
3. **业务逻辑安全测试**: 自动发现应用不变量并生成模糊测试
4. **多 AI 提供商支持**: Anthropic Claude、AWS Bedrock、Google Vertex AI

## 适用场景

- 🏢 安全团队日常渗透测试
- 🔒 CI/CD 集成到开发流程
- 🔍 独立研究人员使用
- ⚠️ 仅限白盒测试 (需要源代码访问权限)
