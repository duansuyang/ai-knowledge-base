# Viora - Desktop-First Agent Runtime

## 1. 项目概述

### 1.1 项目名称
Viora

### 1.2 一句话概述
面向桌面的Agent运行时，支持能力驱动的插件架构。

### 1.3 详细描述
Viora是一个Desktop-first的Agent运行时框架，采用能力驱动的插件架构设计。该项目旨在为桌面应用提供灵活的AI Agent支持，通过插件机制扩展功能。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 桌面优先设计
- **描述**: 专门为桌面应用设计的Agent运行时
- **为什么重要**: 区别于服务器端Agent，更注重本地交互和资源管理

#### 创新点 2: 能力驱动插件架构
- **描述**: 通过能力(Capability)来组织和扩展插件
- **为什么重要**: 提供更灵活的扩展机制

### 3.2 技术特点

| 特点 | 说明 |
|------|-----|
| 语言 | Rust |
| 架构 | 插件化 |
| 目标平台 | 桌面应用 |

### 3.3 应用场景

1. **桌面AI助手**: 为桌面应用添加AI能力
2. **本地Agent运行**: 不依赖云端的Agent执行
3. **插件化扩展**: 通过插件扩展功能
