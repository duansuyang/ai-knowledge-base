# seomachine

## 项目概述

A specialized Claude Code workspace for creating long-form, SEO-optimized blog content for any business.

## 核心功能

### Custom Commands
- `/research` - Keyword research, competitor analysis
- `/write` - Create 2000-3000+ word SEO articles
- `/rewrite` - Update existing content
- `/optimize` - Final SEO audit
- `/publish-draft` - WordPress publishing

### Specialized Agents
- Content Analyzer with SEO scoring (0-100)
- SEO Optimizer
- Meta Creator
- Internal Linker
- Keyword Mapper
- Editor (human-sounding content)
- Performance Analyst
- CRO Analyst

### 高级功能
- Google Analytics 4 & Search Console 集成
- DataForSEO API 集成
- 26 个营销技能 (copywriting, CRO, A/B testing)
- Python 分析模块 (keyword density, readability, SERP analysis)

## 技术亮点

- 51% 减少 cyclomatic complexity (tested on real projects)
- 7-point increase in maintainability index
- 支持 WordPress REST API with Yoast SEO metadata
