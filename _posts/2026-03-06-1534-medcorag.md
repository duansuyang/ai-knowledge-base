# MedCoRAG: Interpretable Hepatology Diagnosis via Hybrid Evidence Retrieval

## 1. 项目概述

### 1.1 项目名称
MedCoRAG: 混合证据检索的可解释肝胆疾病诊断系统

### 1.2 一句话概述
结合多专业共识和RAG的肝胆疾病诊断系统，实现可解释的医学AI诊断。

### 1.3 详细描述
MedCoRAG是一个创新的医学诊断系统，通过多智能体协作和混合证据检索，实现肝胆疾病的可解释诊断。该系统从多个医学来源检索证据，并通过多专业共识机制得出诊断结论。

## 3. 项目优势

### 3.1 核心创新点

**创新点 1: 多专业共识机制**
- 整合不同医学专业的诊断意见
- 提高诊断的准确性和可信度

**创新点 2: 混合证据检索**
- 从多个医学数据库检索证据
- 结构化推理路径，可解释性强

**创新点 3: 迭代式多智能体 deliberation**
- 支持多轮讨论和反思
- 模拟真实医学专家会诊过程

### 3.2 技术优势

| 优势类别 | 具体优势 |
|---------|---------|
| 可解释性 | 完整的诊断推理链 |
| 准确性 | 多专业共识提升正确率 |
| 实用性 | 支持临床决策支持 |

### 3.3 应用场景

- 肝胆疾病辅助诊断
- 医学教育与培训
- 临床决策支持系统

## 4. 总结与评价

MedCoRAG展示了RAG在医疗领域的强大潜力，通过多智能体协作和可解释的推理过程，为医学AI诊断提供了新范式。对于构建可信赖的医疗AI系统具有重要参考价值。
