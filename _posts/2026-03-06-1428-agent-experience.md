# Agent Experience

## 项目概述

### 一句话概述
AI 智能体模式和基础设施的可视化参考指南，涵盖 26 种模式和 5 大类别。

### 详细描述
Agent Experience 是一个开源参考网站，系统整理了 AI 智能体的设计模式、表面交互和基础设施。从基础原语（tool use、ReAct、planning）到生产环境实践，提供全面指南。

#### 创新点 1: 系统化模式分类

## 技术细节

### 核心方法/架构

#### 5大类别

1. **Foundations (基础)**
   - Mental models: tool use, ReAct, planning, reflection
   
2. **Patterns (模式)**
   - Concrete architectures: MCP, computer use, multi-agent orchestration, skills
   
3. **Infrastructure (基础设施)**
   - memory, context management, sandboxes, guardrails, harnesses
   
4. **Surfaces (交互表面)**
   - generative UI, IDE agents, CLI agents, chat interfaces
   
5. **Design (设计)**
   - human-in-the-loop, observability, evaluation
