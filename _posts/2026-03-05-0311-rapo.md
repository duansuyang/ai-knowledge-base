# RAPO: Expanding Exploration for LLM Agents via Retrieval-Augmented Policy Optimization

## 1. Project Overview

### 1.1 一句话概述
提出检索增强策略优化(RAPO)框架，通过引入外部检索信号扩展Agentic RL中的探索空间，解决传统方法仅依赖on-policy探索的局限性。

### 1.2 详细描述
Agentic Reinforcement Learning (Agentic RL) 在 LLM Agent 中展现潜力，但现有方法依赖纯 on-policy 探索，限制了对新推理视角的发现。RAPO 通过检索增强的策略优化，引入外部 off-policy 轨迹进行细粒度 step-level 探索。

## 3. 项目优势

### 3.1 核心创新点

**创新点 1: Hybrid-policy Agentic Rollout**
- 允许 Agent 持续对检索到的 off-policy step-level traces 进行推理
- 动态扩展 Agent 的推理感受野，实现更广泛的探索

**创新点 2: Retrieval-aware Policy Optimization**
- 通过检索奖励和重要性塑形校正策略梯度估计
- 稳定训练并优先考虑检索照亮的探索

## 5. 总结与评价

### 5.1 综合评价
RAPO 是 Agentic RL 领域的重要进展，解决了 LLM Agent 探索受限的问题。通过引入检索增强的探索机制，显著提升了 Agent 在复杂多步任务中的表现。

### 5.2 值得关注的点
1. 创新的 Hybrid-policy 探索策略
2. 检索奖励与策略优化的结合
3. 在多个 benchmark 上的显著提升
