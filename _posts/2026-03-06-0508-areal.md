# AReaL - 大规模异步强化学习系统

## 1. 项目概述

### 1.1 项目名称
AReaL (Asynchronous Reinforcement Learning)

### 1.2 一句话概述
来自清华和蚂蚁集团的开源大规模异步强化学习训练系统，专为大型推理和智能体模型设计。

### 1.3 详细描述
AReaL 是一个完全异步的强化学习训练系统，用于训练大规模的推理和智能体模型。由清华叉院和蚂蚁集团 AReaL 团队开发。基于开源项目 ReaLHF 构建，致力于提供完整的训练细节、数据和基础设施。

- [ ] 多模态 (VLM、视频生成、图像生成)
- [x] 强化学习 (RLHF、GRPO、DPO)
- [ ] 具身智能 (机器人、Sim2Real)

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: 完全异步训练
- **描述**: 实现业界领先的异步 RL 训练速度，支持大规模分布式训练
- **为什么重要**: 大幅提升训练效率，降低计算成本

#### 创新点 2: 多算法支持
- **描述**: 支持 GRPO, PPO, DAPO, REINFORCE, RLOO, LitePPO, DR-GRPO, GSPO 等多种 RL 算法
- **为什么重要**: 满足不同场景需求，便于算法研究和对比

#### 创新点 3: Agentic RL
- **描述**: 原生支持智能体 RL 训练，可与任意智能体框架集成
- **为什么重要**: 推动 AI Agent 训练平民化

### 3.2 技术优势

| 优势类别 | 具体优势 | 量化指标 |
|---------|---------|---------|
| 性能提升 | 异步训练速度 | 业界领先 |
| 效果提升 | 235B MoE 模型超越 GPT-5 | τ²-bench 与 Gemini 3.0 Pro 持平 |
| 扩展性 | 大规模分布式训练 | 支持多节点多GPU |
| 灵活性 | 支持多种模型和算法 | Qwen2/3, Gemma 3, VLM 等 |

### 3.3 应用场景

1. **LLM 推理能力提升**: 数学、代码、搜索等推理任务
2. **智能体训练**: 客服、搜索、多轮对话等 Agent 场景
3. **多模态理解**: VLM 视觉语言模型训练
4. **对齐训练**: RLHF Reward Modeling

## 4. 技术细节

### 4.1 核心方法/架构

#### 4.1.1 异步 RL 训练
基于 ReaLHF 构建的异步训练架构，实现训练与推理分离，最大化 GPU 利用率。

#### 4.1.2 支持的训练后端
- Megatron
- PyTorch FSDP
- PyTorch Archon

#### 4.1.3 支持的推理后端
- vLLM
- SGLang

## 5. 实验流程与结果

### 5.1 主要成果

- **AReaL-SEA-235B-A22B**: 235B MoE 模型，在 τ²-bench 上超越 GPT-5，与 Gemini 3.0 Pro 性能持平
- **数学推理**: GSM8K 等数据集上效果显著
- **代码能力**: 代码生成任务表现优异
- **搜索 Agent**: 端到端搜索 Agent 训练
