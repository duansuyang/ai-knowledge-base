# Vision-Agents

## 1. 项目概述

### 1.1 一句话概述
Stream开源的视觉智能体框架，可快速构建基于任意模型或视频提供商的视觉Agent。

### 1.2 详细描述
Vision-Agents是Stream推出的开源视觉智能体框架，支持使用任意模型或视频提供商快速构建视觉Agent。该框架提供了端到端的视觉理解和推理能力。

### 1.3 所属领域
- [x] Agent/IAG (智能体、工作流、MCP、Tool Use)
- [x] 多模态 (VLM、视频生成、图像生成)
