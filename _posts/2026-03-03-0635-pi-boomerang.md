# pi-boomerang

## 1. 项目概述

### 1.1 项目名称
pi-boomerang

### 1.2 一句话概述
面向pi coding agent的高效token自主任务执行框架，通过上下文压缩实现资源高效利用。

### 1.3 详细描述
该项目提出了一种Token高效的自主任务执行方法，专门针对pi coding agent设计。通过上下文压缩技术，在保持任务执行能力的同时显著降低token消耗。

## 3. 项目优势

### 3.1 核心创新点

#### 创新点 1: Context Collapse上下文压缩
- **描述**: 通过创新的上下文压缩技术，在长对话场景下高效处理历史信息
- **为什么重要**: 长上下文是LLM的痛点，该方案能显著降低token消耗和推理成本
- **对比现有方法**: 传统方法直接传递完整上下文，该方案智能压缩保留关键信息
