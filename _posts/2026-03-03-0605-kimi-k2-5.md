# Kimi K2.5: Visual Agentic Intelligence

## 项目概述

Kimi K2.5是月之暗面（Moonshot AI）开源的原生多模态智能体模型，在约15万亿视觉和文本混合token上进行持续预训练。支持视觉理解、推理、Agent工具调用、思考模式和即时模式。

## 核心架构

| 参数 | 规模 |
|------|------|
| 总参数量 | 1T |
| 激活参数量 | 32B |
| 专家数量 | 384 |
| 每token激活专家数 | 8 |
| 上下文长度 | 256K |
| 词表大小 | 160K |

## 核心特性

### 🖼️ 原生多模态
在视觉-语言token上进行预训练，擅长视觉知识、跨模态推理和基于视觉输入的Agent工具调用。

### 💻 视觉编程
K2.5可根据视觉规格（UI设计、视频工作流）生成代码，并能自主编排工具处理视觉数据。

### 🐝 Agent Swarm
从单智能体扩展到自-directed、协调的群集式执行方案。将复杂任务分解为并行子任务，由动态实例化的领域特定Agent执行。
