# nanoAgent - 极简Agent实现

## 1. 项目概述

### 1.1 项目名称
nanoAgent

### 1.2 一句话概述
仅用100行Python代码实现一个可与系统交互的AI Agent。

### 1.3 详细描述
nanoAgent是一个极简的AI Agent实现，使用OpenAI的Function Calling功能。Agent可以执行bash命令、读取文件和写入文件。代码量极小，是理解Agent本质的最佳入门项目。

## 4. 技术细节

### 4.1 核心方法

Agent使用OpenAI Function Calling的循环：
1. 接收用户任务
2. 决定使用哪些工具（bash, read_file, write_file）
3. 执行工具
4. 返回结果给模型
5. 重复直到任务完成

### 4.2 技术栈

| 类别 | 技术/工具 |
|------|----------|
| 框架 | Python |
| API | OpenAI Function Calling |
