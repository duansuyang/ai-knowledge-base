# Kiwi-Edit: Versatile Video Editing via Instruction and Reference Guidance

## 1. Project Overview

### 1.1 Project Name
Kiwi-Edit

### 1.2 One-sentence Summary
A unified video editing architecture that combines instruction and reference guidance through a scalable data generation pipeline and learnable queries for high-fidelity controllable editing.

### 1.3 Detailed Description
Kiwi-Edit addresses the limitations of current instruction-based video editing methods which struggle with precise visual control. The paper introduces a scalable data generation pipeline that transforms existing video editing pairs into high-fidelity training quadruplets using image generative models. They also built RefVIE, a large-scale dataset, and RefVIE-Bench for evaluation. The Kiwi-Edit architecture uses learnable queries and latent visual features for reference semantic guidance.

- **Description**: Transforms existing video editing pairs into high-fidelity training quadruplets using image generative models
- **Why Important**: Addresses the scarcity of high-quality paired training data for reference-guided editing

## 4. Technical Details

### 4.1 Core Method

#### Method Description
Kiwi-Edit uses a unified architecture that combines:
1. **Learnable Queries**: For capturing editing intentions
2. **Latent Visual Features**: For reference semantic guidance
3. **Progressive Multi-Stage Training Curriculum**: For achieving both instruction following and reference fidelity

#### Data Pipeline
- Transform existing video editing pairs into quadruplets
- Use image generative models to create synthesized reference scaffolds
- Build RefVIE dataset for training
